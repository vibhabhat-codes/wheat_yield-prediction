# 🌾 Wheat Yield Prediction using Nature-Inspired Optimization and Deep Learning

## 📌 Project Overview

This project predicts **winter wheat yield** using machine learning, deep learning, and modern nature-inspired feature selection techniques.
A hybrid ensemble model is built and explainability techniques are applied for interpretability.

---

## 🚀 Pipeline

1. Data preprocessing
2. Baseline ML models
3. Hyperparameter tuning (RandomizedSearchCV)
4. Nature-inspired feature selection

   * Whale Optimization Algorithm (WOA)
   * Harris Hawks Optimization (HHO)
   * Harmony Search (HS)
5. Retraining on selected features
6. Deep learning models

   * MLP
   * TabNet
   * Transformer
   * Graph Neural Network (GNN)
7. Hybrid ensemble model
8. Explainability

   * SHAP
   * LIME
   * Anchor
   * DiCE

---

## 🧠 Models Used

* Random Forest / Gradient Boosting
* MLP

* Graph Neural Network (GCN)

---

## 🎯 Objective

Predict wheat yield using:

* Climate variables
* Vegetation indices

---

## 📊 Evaluation Metrics

* R² Score
* RMSE
* MAE

---

## 🔍 Explainability

The model is interpreted using:

* SHAP (global importance)
* LIME (local explanation)
* Anchor (rule-based explanation)
* DiCE (counterfactual explanation)

---

## 📈 Results

Nature-inspired feature selection improves performance.
Hybrid ensemble model achieves best prediction accuracy.
